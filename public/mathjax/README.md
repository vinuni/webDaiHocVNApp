# MathJax local fallback

Place `tex-mml-chtml.js` here for offline/local fallback when CDNs (unpkg, jsdelivr) fail.

**To add the file:**

1. Download from: https://unpkg.com/mathjax@3/es5/tex-mml-chtml.js
2. Save as `tex-mml-chtml.js` in this folder

Or run (from project root):
```bash
curl -o public/mathjax/tex-mml-chtml.js https://unpkg.com/mathjax@3/es5/tex-mml-chtml.js
```
